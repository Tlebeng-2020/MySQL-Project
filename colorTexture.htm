Choosing Colors and Textures

You can choose colors and textures by using the Solid Color, Gradient, or Texture tabs:

Solid Color
choose a solid color from the Color palette. Click Custom button to open the Color dialog box. The Color dialog box contains additional colors to select from and the option to define your own color.
Gradient
blends from one color to another on your graph. The Preview box shows you how the blend will look. Select a Start color and End color by clicking Click to open the Color dialog box. Also, set the direction of the blend: either horizontally in the direction of the X axis, or vertically in the direction of the Y axis. Use the Reverse button to swap the start and end colors.

Texture
choose a GIF, JPEG, or BMP image to supply the texture. The Preview box shows you the image that is currently selected in the Selection drop-down box. To select an image from your file system, click the ... button, and then choose the image from the Open dialog box. Use the Fit drop-down menu to select Tile or a relative alignment for your image such as Center or Right. The Color fill radio buttons allow you to specify that the image and the color fill are blended or that the image replaces the color fill. Select the Blend radio button to blend the color selected in the Color tab with the image. Select the Replace radio button to specify that the image replaces the color selected in the Color tab.

Note: Each of these tabs may not be available in all dialog boxes. You must click the Apply and OK buttons or the OK button to apply any changes made in any of these tabs to your graph. In addition, you must also click any OK buttons in dialog boxes still opened that were used to open of this dialog box.
