Changing the Color Display of Data Values

The Categories tab allows you to change the colors assigned to map regions or markers and the way response variable data values are represented by colors.

Use... To...
Edit categories for Select a variable display type and view its Categories settings. Marker colors and Region colors will appear in the drop-down list if variables for these variable display types are assigned in the General tab.
Category type Select a color class assignment from the drop-down list. Once selected, the map's appearence and the text boxes in the interval settings box change. If the variable assigned to Region color or Marker color in the General tab is numeric, Continuous, Equal width, Equal counts, and Unique values will appear in the drop-down list. For text variables assigned to Region color and Marker color, Unique values will be the only option in the drop-down list.
Intervals or Gradient limits Change the class intervals for Equal width and Equal count Category types or the gradient limits for the Continuous Category type. The Category type selected either changes this label from Intervals to Gradient types or completely disables this option if neither intervals or gradient limits are applicable to the Category type selection. The Intervals or Gradient limits may be increased or decreased by using the arrows.
Use color ramp Use colors in a range (from least saturated color to most saturated) for regions and markers instead of using unrelated colors--provided there are enough values for a range.
Reset Change the map's appearence and the Categories settings to their settings prior to any alterations made using the Categories tab. The Reset button only resets changes made while the Properties sheet is open. The Reset button is enabled when the Categories settings are changed and is disabled when clicked.

The Region color or Marker color drop-down lists in the General tab must have a variable assigned to enable the Categories tab. The color gradients or color schemes may also be changed using Style Editor... in the pop-up menu.
