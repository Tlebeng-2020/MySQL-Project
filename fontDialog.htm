Font Dialog Box

Use the Font dialog box to change the properties of the text in your graph.
Font
Select the name of the font that you want to use.
Font style
Select a font style.
Size
Select a font size in points.
Color
Select the font color by using the Color dialog box.

The Sample field shows an example selection of text with your font options applied.

Click OK to accept the current options and return to the previous dialog box.
