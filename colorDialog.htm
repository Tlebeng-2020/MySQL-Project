Color Dialog Box

To select a color, do one of the following:
  • Choose a standard color from the Standard: drop-down list.
     
  • Choose a custom color by clicking a color in the Custom: color chart.
     
  • Click the More... button to select a color using one of the following methods
     
    • Specify HSB values (hue, saturation, brightness) by entering numbers in the appropriate boxes, or using the slider bars.
    • Specify RGB values (red, green, blue) by entering numbers in the appropriate boxes, or using the slider bars.

Click OK.
