Using the Color dialog box

The Color dialog box enables you to change the colors that are applied to the map regions, data markers, legends, and background. To display the Color dialog box, select a color box in the Regions, Markers, General, Categories, or Legend tabs of the Properties sheet.

When you display the Color dialog box, the selected color is shown in the Preview box. You can change the color in the Preview box in one of the following ways:

  • Select a color name from the Standard pull-down menu. Click on the down arrow, scroll through the list, and click on a color name.

  • Manually position the pointers in the Custom panel. Click in the Custom panel to choose a color.

  • Specify numeric color values by clicking the More button.

If black is selected in the Standard pull-down menu, you must click the More button and specify a numeric value in the Brightness text box or use the Brightness slide bar to brighten the color value.

Specify numeric values in the following ways:

  • Enter a value from 0 to 255 for Hue, Saturation, and Brightness, or drag the slide bars to select these values.
  • Enter a value from 0 to 255 for the intensity of Red, Green, and Blue, or drag the slide bars to select these values.
